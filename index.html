Hi, I'm

Aman Pathak

I reject abstraction that hides truth. I build from silicon up.

I'm an undergraduate researcher and systems engineer who believes understanding the machine from transistors to userspace is the point - not a means to an end. At BITS Pilani / ENineHQ, I research hardware security for RISC-V architectures: side-channel attack mitigation, formal verification of cache coherency protocols, and ML-based malware analysis on embedded binaries. I've also designed Ku/X-band antennas at KIIT and filed 6 RISC-V patents spanning data-flow integrity monitors, cryptographic session engines, and on-chip ML inference.

I build things that strip away illusion - TCP/IP stacks from scratch, memory allocators that replace glibc, cycle-accurate CPU emulators, container runtimes using raw Linux namespaces and seccomp, bytecode VMs in Go, and neural networks from nothing but matrix math. I write about the craft on my blog, where I've published 58 posts on architecture, compilers, C, and the philosophy of low-level engineering. I also teach - because the knowledge should outlive me.
